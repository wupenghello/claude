# Claude AI Chat

一个基于 Claude API 的智能对话应用，使用 Vue 3 + TypeScript + Vite 构建。

## 功能特点

- 🤖 支持多个 Claude 模型选择
  - Claude 3 Opus：最强大的模型
  - Claude 3 Sonnet：性能价格比最优
  - Claude 3 Haiku：快速响应版本
  - Claude 3.5 系列：最新优化版本

- 💬 对话功能
  - 多会话管理
  - 会话历史保存
  - 支持文本和图片上传
  - Markdown 格式支持
  - 实时输入反馈

- 🎨 界面设计
  - 苹果风格设计
  - 响应式布局
  - 深色/浅色主题
  - 流畅动画效果

- 🛠 技术特性
  - TypeScript 类型支持
  - Vue 3 组合式 API
  - Vite 构建工具
  - 模块化设计
